Ejecucion condicional if

Sentencia if evalula la condición, que se encuentra en los parentesis si el resultado es true, ejecuta las instrucciones de un bloque de codigo

Clausula Else, si la condicion, es falsa entonces ejecuta las instrucciones de else

if, else, else if ?

Una secuencia de operadores de signos de interrogacion ? puede devolver un valor que dependa de mas de una condición

Para buscar literalmente caracteres especiales [ \ ^ $ . | ? * + ( ), se les antepone una barra invertida \ (“escaparlos”). Se debe escapar / si estamos dentro de /.../ (pero no dentro de new RegExp). Al pasar una cadena a new RegExp, se deben duplicar las barras invertidas \\, porque las comillas de cadena consumen una.
