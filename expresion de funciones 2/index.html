Javascript Fuinciones Callback

Funcion Ask(question, yes, no) con 3 argumentos, question: Texto de la pregunta, yes funcion a ejecutar, si la respuesta es si,no: funcion a ejecutar si la respuesta es No. La funcion debera preguntar la question, y dependiendo de la respuesta de usuario, llamar si, no

function Ask(question, yes, no) { if(confirm(question)) yes() else no(); } function showOk(){ alert("estas de Acuerdo. "); } function showCancel(){ alert("Cancelaste la ejecucion "); } ask("Estas de acuerdo ?" ,showOk, showCancel);

Las funciones son declaradas justo dentro del llamado ask(...) No tiene nombre, por lo tanto se denominan, anonimas. Tales funciones no se pueden acceder fuera del ask (porque no estan asignadas a variables)
