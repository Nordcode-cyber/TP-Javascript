JavaScript Comparaciones

Numericos, cadenas, booleanos

En este artículo, aprenderemos más sobre los diferentes tipos de comparaciones y de cómo las realiza JavaScript, incluidas las peculiaridades importantes. Al final, encontrará una buena receta para evitar problemas relacionadas con las “peculiaridades” de JavaScript.

Booleano es el resultado Como todos los demás operadores, una comparación retorna un valor. En este caso, el valor es un booleano. true – significa “sí”, “correcto” o “verdad”. false – significa “no”, “equivocado” o " no verdad".
